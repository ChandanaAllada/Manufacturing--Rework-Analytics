# Manufacturing Rework Analytics

## Project Overview
This project analyzes manufacturing rework data to identify which production lines and defect types are causing the most rework. The goal is to help manufacturing teams reduce waste, improve quality, and increase efficiency.

## Problem Statement
In manufacturing, rework increases cost and delays delivery. Without data analysis, it's hard to know which line or defect type is the biggest issue.

## Approach
1. **Data Collection**: Created sample dataset with Production Line, Defect Type, and Rework Count
2. **Analysis**: Used Python, Pandas to find:
   - Which production line has the highest rework
   - Which defect type occurs most frequently
3. **Visualization**: Used Matplotlib to create a bar chart for easy insights

## Key Findings
- **Highest Rework Line**: Line-B with 80 reworks
- **Most Common Defect**: Soldering with 105 occurrences
- **Insight**: Line-B and Soldering should be prioritized for process improvement

## Technologies Used
- Python
- Pandas for data analysis
- Matplotlib for data visualization
- Jupyter Notebook

## How to Run
1. Install dependencies: `pip install pandas matplotlib`
2. Run all cells in `Rework-Analytics.ipynb`
